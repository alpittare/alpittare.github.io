TECHNICAL SPECIFICATIONS
================================================================================

BACKEND ARCHITECTURE:
- Framework: Convex (serverless backend)
- Language: TypeScript
- Database: PostgreSQL (managed by Convex)
- API: REST/JSON over HTTP
- Deployment: Convex Cloud

CLIENT ARCHITECTURE:
- Language: JavaScript (ES6+)
- No SDK dependency (pure fetch)
- HTTP Client: Custom ConvexClient
- Auth: Client-side with tokens
- Storage: localStorage + Capacitor

DATABASE SCHEMA:
Players Table:
  - ID, name, email, stats, settings
  - Indexes: email, created, lastSeen

Sessions Table:
  - ID, playerId, game, score, difficulty, won, duration, timestamp
  - Indexes: playerId, game, timestamp, playerId+game

Leaderboard Table:
  - ID, playerId, playerName, game, score, difficulty, rank, season
  - Indexes: game, game+difficulty, season, playerId+game

Multiplayer Rooms Table:
  - ID, code, hostId, guestId, game, status, scores, timestamps
  - Indexes: code, hostId, status, game, created

Multiplayer Queue Table:
  - ID, playerId, game, difficulty, status, matchedRoomId, timestamp
  - Indexes: playerId, game, status, game+status

Achievements Table:
  - ID, playerId, achievementId, unlockedAt, game
  - Indexes: playerId, game, playerId+game

Campaign Progress Table:
  - ID, playerId, game, level, stars, bestScore, timestamps
  - Indexes: playerId, game, playerId+game

Seasons Table:
  - ID, name, startDate, endDate, game, active
  - Indexes: game, active

Analytics Events Table:
  - ID, playerId, event, game, difficulty, metadata, timestamp
  - Indexes: playerId, event, timestamp

USAGE QUICK START
================================================================================

1. Install Dependencies:
   npm install

2. Start Development:
   npm run dev

3. Build for Production:
   npm run build

4. Deploy to Production:
   npm run deploy

5. Client Integration:
   const client = new ConvexClient({
     deploymentUrl: 'https://your-deployment.convex.cloud'
   });

   const auth = await new AuthService({ convexClient: client })
     .initialize();

   const analytics = new AnalyticsService({
     convexClient: client,
     playerId: auth.playerId
   });

   // Submit game session
   await client.mutation('submitGameSession', {
     playerId: auth.playerId,
     game: 'cricket',
     score: 150,
     difficulty: 'medium',
     won: true,
     duration: 120
   });
