📊 Platform Overview

The CMI NaaP AI Network Intelligence Platform delivers six specialized AI/ML microservices that work synergistically to provide autonomous network operations, predictive analytics, and intelligent incident management across multi-domain Cisco Service Provider networks. Each service is independently scalable, containerized for Kubernetes deployment, and integrated through a unified REST API.

🚀 Deployment Guide

Docker Compose (Local/Dev)

docker-compose up -d

All 6 services start on ports 8001-8006, dashboard on 8000

Kubernetes (Production)

kubectl apply -f kubernetes/namespace.yaml
kubectl apply -f kubernetes/configmap.yaml
kubectl apply -f kubernetes/deployment.yaml
kubectl apply -f kubernetes/service.yaml
kubectl apply -f kubernetes/ingress.yaml

Full platform deployed to cmi-naap namespace with auto-scaling 2-10 replicas

Monitoring & Logs

# Monitor all services
kubectl logs -n cmi-naap -f deployment/blast-radius-analysis

# Check pod status
kubectl get pods -n cmi-naap

# View metrics
kubectl top pods -n cmi-naap
