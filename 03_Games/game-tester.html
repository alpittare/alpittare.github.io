Game Portfolio Tester

Scan QR codes on your iPhone to test all 6 games on mobile

Setup Instructions (One-Time)

  1. Easiest: Double-click start-server.bat in the Games folder
    It auto-detects Python, Node.js, or falls back to PowerShell
  2. Or manually open Command Prompt and run one of these:
    cd C:\Users\ADITI\Desktop\07_Cluade_Code_folder\03_Games
    Option A — Python:
    python -m http.server 8080
    Option B — Node.js:
    npx http-server -p 8080 --cors
  3. Find your PC's local IP:
    ipconfig | findstr "IPv4"
    Look for something like 192.168.x.x
  4. Make sure your iPhone is on the same WiFi network as your PC
  5. Enter your IP below and click Generate
Port: 8080 (default Python HTTP server)
Pro Tips:
- Games are HTML5 Canvas — they work directly in Safari on iPhone
- For best experience, add to Home Screen (Share → Add to Home Screen) for fullscreen mode
- Touch controls are already built into all games
- If QR doesn't work, check Windows Firewall allows port 8080
